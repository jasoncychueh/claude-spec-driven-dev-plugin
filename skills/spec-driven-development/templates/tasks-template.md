# Tasks Document

## 任務狀態標記

| 標記 | 意義 | 說明 |
|------|------|------|
| `[ ]` | 待執行 | 尚未開始的任務 |
| `[x]` | 已完成 | 已實作並驗證通過 |
| `[~]` | 需重做 | 設計變更影響，需重新實作 |
| `[-]` | 已移除 | 設計變更刪除，程式碼已清理 |
